# Good Catch: Bear and Krill

## Overview
Beau the Bear is preparing for a long winter hibernation. Help him catch enough fish for him and his bear friends before the lake freezes over!

## Gameplay
- Click on fish to collect
- Collect fish before the timer runs out

## Tech Stack
- Unity (3D)
- C#
- Maya (3D models)
